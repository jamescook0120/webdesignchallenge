Summary

Today we are going to look at some weather information. We will be looking at how the weather changes as one approaches the equator. To accomplish this task we used the OpenWeather API to compile data for over 500 cities.

Following this we use MatPlotLib to plot different weather elements in comparison to the latitude. This website will house those comparisons as well as the raw data, and explanations and descriptions of any trends we found.
